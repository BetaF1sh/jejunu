## 길이가 n인 두 유한 수열이 서로 같은지 비교하는 알고리즘 isEqual을 재귀와 비재귀 형식으로 각각 작성하시오.

> 입력 : 수열 ${s(s1, s2 .. sn)}$와 ${t(t1, t2, t3 .. tn)}$, 그 길이 ${n}$
> 출력 : 수열 s와 t가 같을 경우 true, 그렇지 않을 경우 false

### 비 재귀 형식의 알고리즘

```c
bool isEqual(s, t, n) {
    for (i=1; i<=n; i++) {
        if (s[i] != t[i])
            return false
    }
    return true
}
```

### 재귀 형식의 알고리즘

```c
bool isEqual(s, t, n) {
    if (s[n-1] != t[n-1])
        return false
    if (n==1)
        return true
    return isEqual(s, t, n-1)
}
```
